# 프로그래머스 코딩테스트 풀이

이 저장소는 **프로그래머스(Programmers)** 플랫폼의 코딩테스트 문제를 풀이한 소스 코드를 정리한 폴더입니다.  
각 문제별로 **문제 번호**, **문제 이름**, **난이도**, **사용 언어**, **풀이 접근법** 등을 명확히 기록하여, 추후 복습 및 학습 자료로 활용할 수 있도록 구성했습니다.

---

## 📂 폴더 구조
├── level 1/  
│   ├── 1_크레인_인형뽑기_게임.cpp 
│   ├── 2_키패드_누르기.js
│   └── ...  
├── level 2/  
│   ├── 1_전화번호_목록.cpp 
│   └── ...  
├── level 3/  
└── level 4/  
text- `level1`, `level2`, ... : 프로그래머스 문제 레벨별 폴더
- 파일명 형식: `{풀이 순서}_{문제제목}.{확장자}`
  - 예: `1_크레인_인형뽑기_게임.py`

---

## 🛠️ 사용 언어

| 언어       | 확장자 | 비고 |
|------------|--------|----------|
| C++        | `.cpp` | 주로 사용 |
| JavaScript | `.js`  | 일부 문제 |
